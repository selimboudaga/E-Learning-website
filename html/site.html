Introduction

Le web tient sa richesse de par des langages essentiels, structurés et structurants. L’un permet la structure globale d’une page, l’autre sa mise en forme et le dernier son dynamisme.
Le HTML, CSS et JS forment la base pour tout site Web et application Web moderne.



Outils

En tant que développeur Web, les trois principaux langages que nous utilisons pour créer des sites Web sont HTML, CSS et JavaScript.
JavaScript est le langage de programmation, nous utilisons HTML pour structurer le site et nous utilisons CSS pour concevoir et mettre en page la page Web.



Html

Apprenez le HTML , le langage commun à tous les sites Web. HTML décrit la structure de base et le contenu d'une page Web. Si vous souhaitez créer un site Web ou une application Web, vous devez connaître le langage HTML.

Apprenez les bases de CSS, l'une des technologies de base pour la conception et la création de sites Web et d'applications.Peu importe le type de site ou d'application Web que vous souhaitez créer,il faut utiliser CSS

Javascript

JavaScript est un langage de programmation qui pilote le Web, la conception de l'interface utilisateur frontale à la programmation backend côté serveur. Trouverez JavaScript sur chaque site et application Web.
